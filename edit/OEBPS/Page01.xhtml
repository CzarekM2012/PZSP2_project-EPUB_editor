User manual

Welcome to Epub Editor. This is a short manual that will show you around the application. This place on the right where you can see the text is an epub preview. Here you will see the changes you make and how they affect the book. To go to the next page, click a button with an arrow above this preview, or use ctrl + . keyboard shortcut (ctrl + , for the previous page).

If you feel ready to work with the application and do not want to waste more time on the tutorial, start editing your .epub file by opening it. Use File -> Open (ctrl + o) from menu bar. When you are satisfied with the changes, you can save a file with File -> Save (ctrl + s).
